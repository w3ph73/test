== Description ==

Search real estate listing by using the Great Real Estate plugin.

== Installation ==

1. Upload the folder `gre-search` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[gre-search]` in the page that you want to display the search form
